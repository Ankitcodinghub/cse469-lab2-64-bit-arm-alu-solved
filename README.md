The ALU has 8 ports.&nbsp; These ports are the two input ports A and B, the output port, ALU control, zero detect output, overflow detect output, negative detect output, and the carryout output.&nbsp; Like with lab #1, there is a stimulus file provided for you (though it is woefully inadequate..).&nbsp; That file includes comments that state the meaning of each output flag, as well as the values of the ALU control signals and the corresponding function that should be computed.&nbsp; Remember that the turn-in format and design requirements from lab #1 apply to this lab as well.
